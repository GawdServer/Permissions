# Permissions
A simple permission and group managing system.
Permissions are stored in Json format.
GawdPerms supports the wildcard (*) and negative perms (-).
Players can have multiple groups as well.

**Commands:**
* !giveperm (username) (permission) - Give a player permission.
* !takeperm (username) (permission) - Take a players permission.
* !saveperms - Save players and groups.
* !loadperms - Reload the players and groups.

**Permissions:**
* perms.give - Allow access to command !giveperm
* perms.take - Allow access to command !takeperm
* perms.save - Allow access to command !saveperms
* perms.load - Allow access to command !loadperms
